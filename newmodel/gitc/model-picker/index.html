Sketchfab Model Picker

Load models easily in your web applications.
Include SketchfabPicker.js in your page. 

<script type="text/javascript" src="SketchfabPicker.js"></script>

Then call the picker to open a new window. 


var picker = new SketchfabPicker();
picker.pick({
    success: function(model) {
        console.log(model);
    }
});
